BitBlock

O BitBlock foi criado na minha cadeira de Projetos 1 na CESAR School, juntamente com meu grupo, ele foi pensado para resolver o problema de milharees de crianças que possuem TEA, o atraso no desenvolvimento na fala, que afeta uma grande porcentagem desse grupo.

Como funciona?

A lógica é simples: a criança escolhe um cubo de instrumento musical, encaixa na base e o sistema reconhece a posição na hora através de sensores, soltando um loop de som e animações na tela. É tecnologia de verdade usada para inclusão e autonomia de um jeito super lúdico

Tecnologias utilizadas

  • Montagem de circuito de Arduínos
  • C++ para Arduíno
  • Python
  • Arduíno Uno

O código

Para o código, utilizamos toda lógica de funcionamento em um código em Python , o código em C++ do Arduíno ficou apenas responsável por receber os inputs e outputs dos sensores RFID/MRC522.
