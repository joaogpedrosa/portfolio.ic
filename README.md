## 🚀 Projetos em Destaque

### 1. FitPlanner
Uma aplicação em Python executada no terminal, criada para facilitar a organização e o acompanhamento da rotina fitness dos usuários (registro de treinos, exercícios, objetivos pessoais e dados de evolução). 
* **Tecnologias:** Python (manipulação de `.txt`, `.csv` e biblioteca `matplotlib.pyplot` para gráficos).
* **Contexto:** Projeto em equipe para a cadeira de Fundamentos de Programação.

### 2. BitBlock
Projeto voltado para inclusão e autonomia, pensado para auxiliar no desenvolvimento de crianças com TEA (Transtorno do Espectro Autista) com atraso na fala de forma lúdica. O sistema reconhece blocos musicais via sensores RFID e dispara animações e sons.
* **Tecnologias:** Arduino Uno, C++, Python, Sensores RFID/MRC522.
* **Contexto:** Projeto em equipe para a cadeira de Projetos 1.
